# Agent Instructions

- Work one phase at a time.
- Keep changes minimal and scoped.
- Do not redesign, refactor broadly, or add 3D unless explicitly requested.
- Preserve routes, slugs, SEO, Payload schemas, API contracts, and existing features.
- Never commit secrets, build artifacts, logs, node_modules, or archives.
- Always report changed files, validation results, and remaining risks.
- Stop at the first failing validation category.
- Do not make autonomous roadmap decisions.
